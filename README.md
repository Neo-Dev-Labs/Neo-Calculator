**Neo Calculator** replaces your default calculator with a powerful, privacy-focused utility suite. From solving complex math equations to calculating land area—Neo does it all without asking for a single permission.

---

## ✨ Why Choose Neo?

🚫 **Zero Permissions:** No Internet. No Tracking. Your data is yours alone.
🎨 **Unlimited Theming:** Customize the app with *any* Hex Code color.
⚡ **High Precision:** Calculate huge factorials (e.g., 10,000!) instantly.
